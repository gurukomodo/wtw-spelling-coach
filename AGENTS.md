# Project: WTW Spelling Coach
This is a Streamlit application designed for teachers to assess the spelling attempts of Mandarin L1 (ESL) students.

## Stack
- Python 3.12
- Streamlit
- SQLite (database_manager.py)
- CrewAI (spelling_logic.py)

## Rules
- When editing app.py, always keep the teacher refinement text box.
- Do not change the 9 diagnostic groups (g0 to g8).
- Do not delete student assessment data without asking.
